## 1. 변수

변수는 **값을 저장하기 위해 확보한 메모리 공간** 또는 **메모리 공간을 식별하기 위한 이름**입니다.

자바스크립트 엔진이 코드를 평가할 때, 숫자나 연산자(예: 10, 20, +)의 의미를 알고 있어야 하고,

표현식(예: `10 + 20`)을 해석하여 좌변과 우변의 숫자를 메모리에 저장한 후, 연산 결과를 다시 메모리에 저장합니다.

이때 메모리에는 1바이트(8비트) 단위의 메모리 셀들이 존재하며, 각 셀은 고유한 메모리 주소를 갖고 있습니다.

> 핵심: 변수를 사용하면 프로그램 내에서 안전하게 메모리에 있는 값을 참조할 수 있고,
> 
> 
> 해당 값에 이름(식별자)을 붙여 쉽게 관리할 수 있습니다.
> 

---

## 1.1 변수의 주요 개념

- **변수 이름(식별자):**
메모리 내에 저장된 값을 식별하기 위한 이름
- **변수 값:**
변수에 실제 저장되는 데이터

---

## 1.2 변수 선언

- 변수를 사용하려면 반드시 **선언**해야 합니다.
    
    자바스크립트에서는 `var`, `let`, `const` 키워드로 변수를 선언합니다.
    
- 변수 선언의 예시:
    
    ```jsx
    var score; // score라는 변수를 선언
    
    ```
    
- 변수 선언 시 암묵적으로 **undefined**가 할당되어 초기화됩니다.
    
    만약 초기화 없이 사용한다면 이전에 남은 “쓰레기 값”이 저장될 수 있으며,
    
    선언되지 않은 식별자에 접근하면 `ReferenceError`가 발생합니다.
    

---

## 1.3 변수 선언과 호이스팅

- 자바스크립트에서는 **변수 선언은 소스코드의 실제 순차 실행 이전에 먼저 처리**됩니다.
    
    즉, 코드 내 어디에 선언하든 **호이스팅(hoisting)**에 의해 선언문이 “끌어올려진” 후 실행되기 때문에
    
    변수에 접근이 가능합니다.
    
- 예시:
    
    ```jsx
    console.log(score); // 출력: undefined (선언은 되있으나 아직 할당되지 않음)
    var score = 80;
    console.log(score); // 출력: 80
    
    ```
    

---

## 1.4 값의 할당과 재할당

- **할당 시점:**
    
    할당(assignment)은 런타임에 실행됩니다.
    
    선언된 변수에 값을 할당할 때마다 메모리 상에서 새로운 공간에 값이 저장됩니다.
    
- **재할당:**
    
    이미 할당된 변수에 새로운 값을 부여하면 이전의 매핑을 해제하고 새 값과 연결됩니다.
    
- 재할당 예시:
    
    ```jsx
    var score = 80;
    score = 90; // score 변수는 이제 90을 가리킴
    
    ```
    
- **참고:**
    
    만약 값의 재할당이 불가능하다면 해당 변수는 상수(constant)입니다.
    

---

## 1.5 식별자 네이밍 규칙

- **사용 가능한 문자:**
    
    특수문자(일부 제외), 알파벳, 숫자, 언더바(`_`), 달러(`$`)
    
- **시작 문자:**
    
    식별자는 문자, 언더바, 혹은 달러로 시작해야 하며 숫자로 시작할 수 없습니다.
    
- **예약어:**
    
    자바스크립트의 예약어는 식별자로 사용할 수 없습니다.
    
- **여러 변수 선언:**
    
    쉼표(`,`)를 사용하여 한 번에 여러 변수를 선언할 수 있습니다.
    
- **네이밍 스타일 예시:**
    
    ```jsx
    var firstName;    // 카멜 케이스 : 주로 변수나 함수에 사용
    var first_name;   // 스네이크 케이스
    var FirstName;    // 파스칼 케이스 : 생성자 함수나 클래스 이름에 사용
    
    ```

## 2. 표현식과 문

프로그램은 **표현식(expression)**과 **문(statement)**으로 구성됩니다.

---

## 2.1 값과 리터럴

- **값(Value):**
    
    표현식이 평가되어 생성된 결과.
    
    예를 들어, `var sum = 10 + 20;`에서 `10 + 20`이 평가되어 `30`이라는 값이 생성되고,
    
    그 값이 `sum`에 할당됩니다.
    
- **리터럴(Literal):**
    
    사람이 이해할 수 있는 문자나 약속된 기호를 사용해 값을 생성하는 표기법.
    
    정수, 소숫점, 2진수, 16진수 등 다양한 형식의 리터럴이 있습니다.
    

---

## 2.2 표현식(Expression)

- **정의:**
    
    표현식은 **평가되어 값으로 변환될 수 있는 문**입니다.
    
- **예시:**
    
    ```jsx
    var score = 50 + 50;  // 50 + 50이 평가되어 100이 생성됨
    score;                // 변수 참조도 표현식 → 100 반환
    
    var x = 1 + 2;        // x는 3
    x + 3;                // 3 + 3이 평가되어 6
    
    ```
    
- **핵심:**
    
    “값으로 평가될 수 있는 문”은 모두 표현식입니다.
    

---

## 2.3 문(Statement)

- **정의:**
    
    문은 프로그램을 구성하는 **기본 실행 단위**이며, 여러 토큰(token)으로 이루어져 있습니다.
    
- **특징:**
    - 각 문(statement)은 보통 세미콜론(`;`)으로 끝납니다.
    - 코드 블록(`{}`) 내에서는 세미콜론이 없어도 ASI(자동 세미콜론 삽입)에 의해 처리됩니다.
- **표현식인 문 vs 표현식이 아닌 문:**
    - 예) `x = 100` → 할당문은 표현식으로 평가됨
    - 예) `var x;` → 변수 선언문은 표현식이 아님

---

## 3. 데이터 타입

자바스크립트의 데이터 타입은 **원시타입(primitive)**과 **객체타입(object)**으로 구분됩니다.

---

## 3.1 원시타입 vs 객체타입

- **원시타입 (Primitive Types):**
    
    숫자, 문자열, 불리언, `undefined`, `null`, `Symbol` 등이 있음.
    
- **객체타입 (Object Types):**
    
    객체, 함수, 배열 등 복합적인 데이터 구조를 포함합니다.
    

---

## 3.2 숫자 타입

- **특징:**
    - 모든 수는 **실수**로 표현되며, 별도의 `int`, `double` 타입은 존재하지 않습니다.
    - 2진수, 8진수 등을 사용할 수 있으나, 내부적으로 10진수로 해석됩니다.
    - 예를 들어, `3 / 2`는 `1.5`와 같이 실수 결과를 반환합니다.
    - 특수 값: `Infinity`, `Infinity`, `NaN` (연산 불가능한 경우)

---

## 3.3 문자열 타입

- **표현 방법:**
    
    문자열은 **작은 따옴표(`'...'`)**, **큰 따옴표(`"..."`)**, **백틱(템플릿 리터럴, ``...``)**으로 감싸서 표현합니다.
    
- **주의:**
    
    문자열을 감싸지 않고 작성하면 공백이나 줄바꿈이 올바르게 인식되지 않습니다.
    

---

## 3.4 템플릿 리터럴

- **템플릿 리터럴의 특징:**
    - 백틱(```)을 사용하여 문자열을 감쌉니다.
    - 줄바꿈과 공백이 그대로 유지되며, 런타임 시 일반 문자열로 변환됩니다.
    - 이스케이프 시퀀스 사용 가능:`\\0` (null), `\\b` (백스페이스), `\\n` (개행), `\\t` (탭) 등.
- **예제:**
    
    ```jsx
    // 일반 문자열
    var str1 = "Hello, World!";
    var str2 = 'Hello, World!';
    
    // 템플릿 리터럴
    var name = "Copilot";
    var message = `Hello, ${name}!
    이 메시지는 줄바꿈과 공백이 그대로 적용됩니다.
    이스케이프 예: \\n, \\t 등을 사용할 수 있습니다.`;
    
    console.log(message);
    
    ```
