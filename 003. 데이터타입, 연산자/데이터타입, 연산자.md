## 데이터 타입

자바스크립트가 값을 저장하고 다루기 위해 사용하는 다양한 데이터 타입에 대해 알아봅니다.

### 1.1 불리언 타입 (Boolean)

- **정의:**
논리적 참(`true`)과 거짓(`false`)만을 나타내는 타입.
- **주요 사용:**
조건문, 반복문 등 제어문에서 조건 판별에 사용됨.

### 1.2 undefined 타입

- **정의:**
오직 하나의 값 `undefined`만을 가짐.
- **특징:**
    - 값이 할당되지 않은 변수에서 기본적으로 나타남.
    - 실수로 값이 없음을 나타내는 용도로 사용하면 혼동을 줄 수 있으므로 권장되지 않음.

### 1.3 null 타입

- **정의:**
오직 하나의 값 `null`만을 가짐.
- **주요 사용:**
    - 변수가 의도적으로 "값이 없음"을 나타내기 위해 사용함.
    - 함수가 유효한 값을 반환할 수 없을 때 `null`을 반환할 수 있음.

### 1.4 심벌 타입 (Symbol)

- **정의:**
변경 불가능한 원시 타입 값으로, 객체의 유일한 프로퍼티 키를 만들기 위해 사용됨.
- **생성:**`Symbol()` 함수를 호출하여 생성하며, 중복되지 않는 유일한 값을 보장함.

### 1.5 객체 타입 (Object)

- **정의:**
자바스크립트에서 가장 중요한 타입이며, 함수, 배열 등 거의 모든 것이 객체의 형태를 띔.
- **특징:**
복합적 데이터와 메서드를 담을 수 있으며, 프로퍼티와 메서드라는 키와 값의 쌍으로 구성됨.

### 1.6 데이터 타입이 필요한 이유

- **메모리 관리:**
    - **값을 저장:** 값이 메모리에 저장될 때, 자료형에 따라 필요한 메모리 공간의 크기를 결정함.
    - **값을 참조:** 값을 읽어올 때, 몇 비트의 메모리를 한꺼번에 읽어와야 하는지 결정함.
    - **해석:** 메모리에 저장된 2진수를 어떤 타입으로 해석할지를 알려줌.
- **예시:**
    
    ```jsx
    let score = 100; // 숫자타입의 값이 할당되어 JS 엔진은 score를 숫자로 해석.
    
    ```
    

### 1.7 동적 타이핑 (Dynamic Typing)

- **개념:**
    - 자바스크립트는 변수 선언 시에 데이터 타입을 명시하지 않으며, **할당된 값**에 의해 타입이 결정됨.
    - 변수의 타입은 **재할당**에 의해 언제든지 변경될 수 있음.
- **정적 타입 언어와의 차이:**
    - **정적 타입 언어:** C, Java처럼 변수 선언 시 타입을 지정하며 그 타입만을 허용함.
    - **동적 타입 언어:** 자바스크립트처럼 실행 시 할당된 값에 의해 타입이 변화할 수 있음.
- **주의사항:**
    - 변수를 꼭 필요한 경우에만 사용하며, 유효 범위를 좁혀 사용합니다.
    - 전역변수의 사용은 최소화하고, 상수를 적극 활용하며, 이해하기 쉬운 변수명을 사용합니다.

## 연산자

연산자는 하나 이상의 피연산자에 대해 특정 **연산(계산)**을 수행하여 새로운 값을 반환하는 **표현식**입니다. 각 연산자의 특성과 사용법을 살펴봅니다.

### 2.1 산술 연산자

- **특징:**
피연산자를 연산하여 새로운 값을 생성하며, 연산이 불가능한 경우 `NaN`을 반환함.
- **이항 산술 연산자 (두 피연산자):**
    - **`+`**: 덧셈 – 단, 피연산자 중 하나가 문자열이면 문자열 연결 연산자로 동작
    - : 뺄셈
    - : 곱셈
    - **`/`**: 나눗셈
    - **`%`**: 나머지
- **단항 산술 연산자 (하나의 피연산자):**
    - **`++`**: 1만큼 증가 (전위, 후위 구분하며 부수 효과가 있음)
    - **`-`**: 1만큼 감소 (전위, 후위 구분)
    - **단항** : 부호 반전
    - **단항 `+`**: 숫자 타입이 아닌 피연산자를 강제로 숫자로 형 변환
- **문자열 연결 연산자 (`+`):**
    - 숫자와 문자열 결합 시 숫자를 문자열로 강제 변환하여 연결함.
    - 예시:
        
        ```jsx
        console.log("1" + 2);      // "12"
        console.log(1 + 2);        // 3
        console.log(1 + true);     // 1 + 1 = 2 (불리언 true는 숫자 1로 변환)
        console.log(1 + false);    // 1 + 0 = 1
        console.log(1 + null);     // 1 + 0 = 1
        
        ```
        

### 2.2 할당 연산자

- **기본 개념:**
변수에 값을 할당하며, 할당문 자체가 표현식으로 평가되어 할당된 값이 반환됨.
- **기본 연산자:**
    - **`=`**
- **복합 할당 연산자:**
    - **`+=`**
    - **`=`**
    - **`=`**
    - **`/=`**
    - **`%=`**
- **예시:**
    
    ```jsx
    let x;
    x = 10;           // x에 10 할당
    x += 5;           // x = x + 5; 결과적으로 15
    // 여러 변수에 동시에 할당:
    let a, b, c;
    a = b = c = 0;    // 오른쪽에서 왼쪽 순으로 평가됨
    
    ```
    

### 2.3 비교 연산자

- **목적:**
    
    두 피연산자를 비교한 후, 결과를 불리언 값(`true` 또는 `false`)으로 반환함.
    
- **동등/일치 비교:**
    - **`==`**: 동등 비교 (값만 같으면 true, 암묵적 타입 변환 발생)
        - 예: `5 == '5' // true`
    - **`===`**: 일치 비교 (값과 타입이 모두 같아야 true)
        - 예: `5 === '5' // false`
    - **`!=`**: 부동등 비교 (값이 다르면 true)
    - **`!==`**: 불일치 비교 (값과 타입 모두 고려)
- **특수 예외:**
    - `NaN`은 자기 자신과도 같지 않으므로, 비교 시에는 `Number.isNaN()` 또는 `Object.is()`를 사용.
- **대소 관계 비교 연산자:**
    - **`>`**
    - **`<`**
    - **`>=`**
    - **`<=`**

### 2.4 삼항 조건 연산자

- **문법:**`조건식 ? 조건식이 true일때 반환 값 : 조건식이 false일때 반환 값`
- **특징:**
조건식의 평가 결과가 불리언이 아니면 강제로 변환됨.
이 연산자 자체가 하나의 **값**을 반환하므로 다른 표현식에 포함될 수 있음.
- **예시:**
    
    ```jsx
    let x = 7;
    let result = (x % 2 === 0) ? '짝수' : '홀수';
    console.log(result); // "홀수" 출력
    
    ```
    

### 2.5 논리 연산자

- **주요 연산자:**
    - **`||` (OR):** 하나라도 true이면 true
    - **`&&` (AND):** 모두 true여야 true
    - **`!` (NOT):** 피연산자의 불리언 값을 반전시킴
- **특징:**
항상 불리언 값을 반환함.
- **예시:**
    
    ```jsx
    console.log(!0);       // true (0은 false로 간주)
    console.log(!'hello'); // false (빈 문자열이 아니라 true)
    
    ```
    

### 2.6 쉼표 연산자

- **개념:**
여러 표현식을 하나의 문으로 연결하며, 왼쪽부터 순차적으로 평가해 마지막 표현식의 결과를 반환함.
- **예시:**
    
    ```jsx
    let x = (1, 2, 3);
    console.log(x); // 3 출력
    
    ```
    

### 2.7 그룹 연산자

- **용도:**
괄호 `()`를 사용하여 기본 연산자 우선순위를 변경하거나 특정 부분을 먼저 평가하도록 함.
- **예시:**
    
    ```jsx
    let result = (2 + 3) * 4; // 괄호 안의 2+3이 먼저 계산되어 5*4=20
    
    ```
    

### 2.8 typeof 연산자

- **기능:**
피연산자의 데이터 타입을 문자열로 반환함.
- **예시:**
    
    ```jsx
    console.log(typeof "");       // "string"
    console.log(typeof 100);      // "number"
    console.log(typeof true);     // "boolean"
    console.log(typeof undefined);// "undefined"
    // 주의: typeof null 은 "object"를 반환하는데, 이는 오래된 버그입니다.
    
    ```
    

### 2.9 지수 연산자

- **문법:**`*` 연산자로, 좌항을 밑, 우항을 지수로 하여 거듭제곱 계산을 수행함.
- **예시:**
    
    ```jsx
    console.log(2 ** 2);  // 4
    console.log(2 ** 0);  // 1
    console.log((-5) ** 2); // 25 (음수는 반드시 괄호로 묶어야 함)
    
    ```
    

### 2.10 연산자의 부수 효과 (Side Effects)

- **정의:**
보통의 연산자는 다른 코드에 영향을 주지 않고 값을 반환하지만, 일부 연산자(예: 할당 연산자 `=`, 단항 증가/감소 `++`, `-`, `delete` 등)는 **부수 효과**를 일으켜 변수나 객체의 값을 직접 변경함.
- **예시:**
    
    ```jsx
    let count = 5;
    console.log(count++); // 출력: 5, 이후 count는 6
    
    ```
    

### 2.11 연산자 우선순위

- **개념:**
여러 연산자가 함께 사용될 때 어떤 순서로 평가되는지 결정하는 규칙.
- **조절 방법:**
괄호 `()`를 사용하여 의도한 우선순위를 명확하게 지정.
- **예시:**
    
    ```jsx
    let result = 2 + 3 * 4;      // 곱셈(*)가 덧셈(+)보다 우선 -> 2 + (3*4) = 14
    let result2 = (2 + 3) * 4;   // 괄호로 우선순위 조절 -> 5 * 4 = 20
    
    ```
